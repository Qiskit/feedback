1. Intro: Effective Dimension and Local Effective Dimension¶

2 source papers:

  1. The Power of Quantum Neural Networks, by Abbas et al. (2020)
  2. Effective Dimension of Machine Learning Models, by Abbas et al. (2021)

Effective Dimension:¶

  • Metric based on information geometry: "How many active parameters does the model have?"
  • Related to Expressibility and Generalization ability
  • Valid for classical and quantum NNs

Local Effective Dimension:¶

  • Measure of capacity based on Effective Dimension
  • Key difference: 1 parameter set sample vs. full parameter space (weights)
  • Generalization ability for Trained Models: performance of trained models on tasks involving new data

2. Algorithm Overview¶

  • Step 1: Define an input/weight/data sample space
  • Step 2: Run Algorithm --> Fisher Information Matrix

2. Algorithm Overview¶

  1. Monte Carlo simulation: for pairs of input and weight samples.
  2. Fisher Info. Matrix Computation
  3. Fisher Info. Matrix Normalization: averaging over all input samples and dividing by the matrix trace
  4. Effective Dimension Calculation: according to the formula from Abbas et al.:

# parity maps bitstrings to 0 or 1
def parity(x):
    return "{:b}".format(x).count("1") % 2

output_shape = 2  # corresponds to the number of classes, possible outcomes of the (parity) mapping.
# declare quantum instance
qi_sv = QuantumInstance(Aer.get_backend("aer_simulator_statevector"))

    qc,
    input_params=feature_map.parameters,
    weight_params=ansatz.parameters,
    interpret=parity,
    output_shape=output_shape,
    sparse=False,
    quantum_instance=qi_sv,
)
